# YucoxSSH

A modern SSH client for macOS and Windows.

Official Website: [https://www.yucox.com](https://www.yucox.com)

## Features

- **Modern UI** - Clean, themeable interface with dark mode support
- **Session Management** - Organize SSH sessions with folders
- **Multi-tab Terminal** - Connect to multiple servers in tabs
- **Secure Credential Storage** - Keychain integration on macOS
- **Cross-platform** - Native support for macOS (Universal Binary) and Windows

## Download

### macOS

Download `YucoxSSH-{version}.dmg` from [Releases](../../releases) and drag to Applications folder.

Supports both Intel (x86_64) and Apple Silicon (arm64) Macs.

Minimum version: macOS 10.15

### Windows

Download `YucoxSSH-{version}-windows-x64.zip` from [Releases](../../releases), extract and run `YucoxSSH.exe`.

Minimum version: Windows 7 (64-bit)

## Quick Start

1. Launch YucoxSSH
2. Click **File > New Session** or press `Ctrl+N` (Windows) / `Cmd+N` (macOS)
3. Enter connection details:
   - Session name (optional)
   - Host address
   - Port (default: 22)
   - Username
   - Password or private key
4. Click **Connect**

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + N` | New Session |
| `Ctrl/Cmd + Q` | Quit |
